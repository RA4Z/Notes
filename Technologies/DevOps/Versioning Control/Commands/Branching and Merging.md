[[Git]] commands that facilitate working with branches, isolating development efforts and integrating changes

- `git branch`: Lists, creates, or deletes branches
- `git switch`: Switches to a different branch or restores files
- `git checkout`: Switches branches, restores files, or navigates commit history
- `git merge`: Integrates changes from one branch into another
- `git rebase`: Reapplies commits from one branch onto another, creating a linear history
