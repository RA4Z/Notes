Python is a powerful programming language that can be used in [[Web Development]], [[Automation]], [[Data Analysis]], [[Machine Learning]] and other things.

It's possible to download Python in [Python.org](https://www.python.org/)

To see the installed Python version you can run the following command in terminal:
```Terminal
python --version
```

Python is an interpreted language, what means it's possible to run commands directly in the terminal. For that, open the terminal and write ```python``` to access the interactive interpreter. Then you can write commands similar to:
```python
print("Hello, World!)
```

Python have lots of packages, where can be seen in [[PyPi]] they can be installed in the terminal using the following command:
```Terminal
{python-path} -m pip install {package-name}
```

There are some Python [[Packages]] listed in this documentation.
