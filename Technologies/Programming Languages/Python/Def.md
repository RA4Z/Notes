In Python, functions are defining using the `def` keyword

#### Example: Basic Function
```Python
def greet(name):
   print(f"Hello, {name}!")
greet("Alice") # Output: Hello, Alice!
```

#### Example: Function with Return Value
```Python
def add_numbers(a, b):
   return a + b
result = add_numbers(5, 3)
print(result) # Output: 8
```
