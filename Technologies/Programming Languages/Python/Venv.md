`venv` is a built-in [[ğŸPython]] module that allows developers to create **virtual environments**. A virtual environments is an isolated workspace where you can install [[ğŸPython]] [[Packages]] and dependencies specific to a project, without affecting the global [[ğŸPython]] installation or other projects

```Powershell
# Syntax
python -m venv <env_name>

# Example

```
