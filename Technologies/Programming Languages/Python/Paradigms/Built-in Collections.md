[[üêçPython]] offers several built-in collection data types for storing and organizing data

- **[[Lists]]:** Ordered, mutable sequences of heterogeneous elements
```Python
my_list = [1, "hello", 3.14, True]
```

- **[[Tuples]]:** Ordered, immutable sequences of heterogeneous elements
```Python
my_tuple = (1, "hello", 3.14, True)
```

- **[[Sets]]:** Unordered collections of unique, hashable elements. Sets do not allow duplicate entries
```Python
my_set = {1, 2, 3, 2, 4}
```

- **[[Dictionaries]]:** Unordered mappings of unique, hashable keys to values
```Python
my_dict = {"name": "Alice", "age": 30, "city": "New York"}
```
